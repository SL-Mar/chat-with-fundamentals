\preface

This monograph documents the quantitative methods implemented in the \emph{Chat with Fundamentals} platform, a comprehensive financial analysis system combining database-driven architecture with modern machine learning techniques. The platform processes over 30 years of historical market data for thousands of securities, enabling sophisticated risk analysis, performance attribution, and alpha generation.

\section*{Motivation}

Traditional financial analysis tools often suffer from:
\begin{itemize}
    \item Limited historical data (5-10 years at most)
    \item Slow API-dependent queries
    \item Black-box algorithms with poor interpretability
    \item Fragmented workflows across multiple platforms
\end{itemize}

The Chat with Fundamentals platform addresses these limitations through:
\begin{enumerate}
    \item \textbf{Database-first architecture:} PostgreSQL + TimescaleDB with 30+ years of cached data
    \item \textbf{Transparent methodologies:} All quantitative methods documented with mathematical foundations
    \item \textbf{Unified interface:} From data ingestion to portfolio optimization in a single platform
    \item \textbf{Modern tech stack:} FastAPI, React, PostgreSQL, Redis, Python scientific stack
\end{enumerate}

\section*{Audience}

This monograph is intended for:
\begin{itemize}
    \item \textbf{Quantitative analysts} implementing risk models and performance metrics
    \item \textbf{Data scientists} building machine learning pipelines for alpha generation
    \item \textbf{Portfolio managers} seeking to understand the mathematical foundations of optimization algorithms
    \item \textbf{Students} in financial engineering, computational finance, or quantitative finance programs
    \item \textbf{Software engineers} building financial analysis platforms
\end{itemize}

\textbf{Prerequisites:} Undergraduate-level statistics, linear algebra, and programming experience (Python preferred).

\section*{Structure}

The monograph is organized into ten parts:

\begin{description}
    \item[Part I: Foundational Concepts] Time series analysis, return calculations, data infrastructure
    \item[Part II: Risk Metrics] Volatility modeling (EWMA, CVaR), risk-adjusted performance (Sharpe, Sortino, Calmar), factor models (CAPM, beta/alpha)
    \item[Part III: Simulation \& Forecasting] Monte Carlo methods with Geometric Brownian Motion, cumulative return analysis
    \item[Part IV: Technical Analysis] Trend indicators, momentum oscillators, volatility indicators, technical screening
    \item[Part V: Feature Engineering] Fundamental factors, momentum factors, cross-sectional normalization, PCA
    \item[Part VI: Machine Learning] Factor mining with genetic programming, LSTM/XGBoost/Transformers/GAT models, ensemble methods
    \item[Part VII: Portfolio Optimization] Modern Portfolio Theory, Hierarchical Risk Parity, Black-Litterman model
    \item[Part VIII: Backtesting] Historical simulation, cross-validation, regime analysis
    \item[Part IX: Real-Time Systems] Intraday data processing, order execution algorithms, risk management
    \item[Part X: Advanced Topics] Alternative data (news sentiment, social media), ESG integration, optimization algorithms
\end{description}

\chapter{Glossary of Terms}
\begin{description}
    \item[Alpha] Excess return above the expected return from a factor model (e.g., CAPM)
    \item[Beta] Sensitivity of an asset's returns to market returns
    \item[Sharpe Ratio] Risk-adjusted return metric: $(R - R_f) / \sigma$
    \item[EWMA] Exponentially Weighted Moving Average
    \item[CVaR] Conditional Value at Risk (Expected Shortfall)
    \item[LSTM] Long Short-Term Memory neural network architecture
    \item[GBM] Geometric Brownian Motion
    \item[HRP] Hierarchical Risk Parity portfolio construction method
    \item[MPT] Modern Portfolio Theory
\end{description}
